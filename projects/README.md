# Your Projects

This is where your writing lives. Each project gets its own folder.

## Starting a new project

In Claude Code, type: `/project:newproject`

## Importing an existing manuscript

Have a .docx or .txt file? Type: `/project:import`

## Adding files manually

Create a folder inside `projects/` with your project name and drop your files in:

```
projects/
  my-novel/
    chapters/
      01-chapter-one.md
      02-chapter-two.md
    characters/
      protagonist.md
    notes/
      ideas.md
    outline.md
    style-guide.md
```

The AI adapts to whatever structure you use.
